Hotel Booking Website: Enable customers to browse, select, and book hotel rooms seamlessly while ensuring secure, efficient operations

MVP (No document verification flow)
Clean MVP (Only Essentials + Room Number Added)
User
- id                BIGSERIAL PRIMARY KEY
- name              VARCHAR(100) NOT NULL
- email             VARCHAR(150) UNIQUE NOT NULL
- password          VARCHAR(255) NOT NULL
- role              VARCHAR(20) CHECK (role IN ('ADMIN','MANAGER','CUSTOMER'))
- created_at        TIMESTAMP DEFAULT CURRENT_TIMESTAMP
Hotel
- id                BIGSERIAL PRIMARY KEY
- name              VARCHAR(150) NOT NULL
- location          VARCHAR(255) NOT NULL
- description       TEXT
- manager_id        BIGINT REFERENCES User(id)
- created_at        TIMESTAMP DEFAULT CURRENT_TIMESTAMP
Room
- id                BIGSERIAL PRIMARY KEY
- hotel_id          BIGINT REFERENCES Hotel(id)
- room_number       VARCHAR(20) NOT NULL   -- 🔥 critical addition
- type              VARCHAR(50)            -- e.g. DELUXE, STANDARD
- price             DECIMAL(10,2) NOT NULL
- capacity          INT NOT NULL
- status            VARCHAR(20) CHECK (status IN ('AVAILABLE','BOOKED','MAINTENANCE'))
Booking
- id                BIGSERIAL PRIMARY KEY
- reservation_number VARCHAR(50) UNIQUE NOT NULL
- user_id           BIGINT REFERENCES User(id)
- room_id           BIGINT REFERENCES Room(id)   -- 🔥 direct mapping to actual room
- check_in          DATE NOT NULL
- check_out         DATE NOT NULL
- total_price       DECIMAL(10,2) NOT NULL
- status            VARCHAR(20) CHECK (status IN ('CONFIRMED','CANCELLED','COMPLETED'))
- created_at        TIMESTAMP DEFAULT CURRENT_TIMESTAMP
Review
- id                BIGSERIAL PRIMARY KEY
- user_id           BIGINT REFERENCES User(id)
- hotel_id          BIGINT REFERENCES Hotel(id)
- rating            INT CHECK (rating BETWEEN 1 AND 5)
- comment           TEXT
- created_at        TIMESTAMP DEFAULT CURRENT_TIMESTAMP
Relationships (MVP)
User (1) → (M) Hotel
User (1) → (M) Booking
User (1) → (M) Review
Hotel (1) → (M) Room
Room (1) → (M) Booking
Hotel (1) → (M) Review
